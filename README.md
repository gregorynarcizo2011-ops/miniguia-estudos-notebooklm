Resumo
A Inteligência Artificial é uma tecnologia usada para ajudar computadores a resolver problemas, aprender com informações e executar diferentes tarefas. Ela já faz parte do nosso cotidiano e tende a crescer cada vez mais.
Glossário
IA: Inteligência Artificial.
Algoritmo: sequência de passos para resolver um problema.
Machine Learning:forma de ensinar o computador a aprender com dados.
Rede Neural: sistema inspirado no funcionamento do cérebro humano.
